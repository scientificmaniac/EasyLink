# 个人通讯录管理系统

## 功能介绍

这是一个基于Python和Tkinter开发的个人通讯录管理系统，采用面向对象设计方法，具有以下核心功能：

### 1. 基本信息管理
- 添加联系人：包括姓名、电话、邮箱、备注等信息
- 修改联系人：可以修改已添加联系人的所有信息
- 删除联系人：支持单个联系人删除
- 标记常用联系人：可以将联系人标记为常用，方便快速查找

### 2. 查询功能
- 按姓名模糊搜索：支持部分匹配
- 按电话精确查找：支持完整电话号码匹配
- 分类查看：支持"全部联系人"和"常用联系人"标签页切换

### 3. 数据存储
- 使用JSON格式保存在本地文件(`contacts.json`)
- 程序启动时自动加载数据
- 关闭时自动保存数据

### 4. Excel导出功能
- 支持将联系人信息导出为Excel表格
- 导出文件名为`contacts.xlsx`

### 5. GUI界面
- 使用tkinter设计友好的图形界面
- 包含输入框、按钮、列表框等组件
- 标签页分类显示不同联系人组
- 详情面板实时显示选中联系人信息

## 运行要求

- Python 3.6+
- 依赖库：`openpyxl`（用于Excel导出功能）

## 安装与运行

1. 安装依赖库（如果尚未安装）：
   ```bash
   pip install openpyxl
   ```

2. 运行程序：
   ```bash
   python contact_management.py
   ```

## 使用说明

### 添加联系人
1. 点击"添加联系人"按钮
2. 在弹出的对话框中填写联系人信息
3. 点击"添加"按钮保存

### 修改联系人
1. 在联系人列表中选择要修改的联系人
2. 点击"修改联系人"按钮或双击联系人项
3. 在弹出的对话框中修改信息
4. 点击"保存"按钮更新

### 删除联系人
1. 在联系人列表中选择要删除的联系人
2. 点击"删除联系人"按钮
3. 在确认对话框中点击"是"删除

### 标记常用联系人
1. 在联系人列表中选择要标记的联系人
2. 点击"标记为常用"按钮
3. 联系人会在列表中显示"[常用]"标识

### 搜索联系人
1. 在搜索框中输入关键词
2. 点击"按姓名搜索"或"按电话搜索"
3. 列表中会显示匹配的联系人
4. 点击"重置"按钮恢复显示所有联系人

### 导出Excel
1. 点击"导出Excel"按钮
2. 联系人信息会导出到当前目录下的`contacts.xlsx`文件

## 项目结构

```
contact_management.py
├── Contact类：联系人数据模型
├── DataStorage类：数据存储和加载
├── ContactManager类：联系人管理业务逻辑
├── ExcelExporter类：Excel导出功能
└── ContactGUI类：GUI界面设计
    ├── AddContactDialog类：添加联系人对话框
    └── EditContactDialog类：修改联系人对话框
```
